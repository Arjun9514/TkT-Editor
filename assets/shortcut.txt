File :
Ctrl + o : Open an existing file
Ctrl + n : Open a new file
Ctrl + s : Save a file
Ctrl + Shift + S : Save As
F7 : Spell check
Ctrl + l : Clear screen

Edit :
Ctrl + b : Bold
Ctrl + i : Italics
Ctrl + u : Underline
Ctrl + r : StrikeThrough
Ctrl + Shift + B : Bold & Italics
Ctrl + Shift + R : Remove emphasis
Ctrl + e : Highlight text
Ctrl + f : Find & replace

Help :
F1 : Shortcuts

Print :
Ctrl + p : Generate PDF
